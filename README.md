## 🚀 Tecnologias utilizadas (Frontend do Portfólio)
Este site foi desenhado utilizando o conceito de interface em **Glassmorphism** (Efeito Vidro) aplicados sobre temas escuros (Dark Mode).

- **React.js** (Arquitetura reativa e Componentização)
- **React Bootstrap** (Sistema de Grid UI responsivo)
- **CSS3 Vanilla** (Variáveis, Transições suaves e Backdrop-filters)
- **Devicon** (Mapeamento de ícones e logotipos flexíveis da Stack)

## ⚙️ Como executar o projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em seu ambiente.

1. Instale as dependências executando na raiz do projeto:
```bash
npm install
```

2. Inicialize o servidor local:
```bash
npm start
```

3. O projeto será executado automaticamente. Caso a aba não abra, acesse `http://localhost:3000` em seu navegador.
